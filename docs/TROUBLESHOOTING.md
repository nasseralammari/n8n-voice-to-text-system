# دليل حل المشاكل

## مشاكل وعامة

### المشكلة: التطبيق لا يبدأ

**الأعراض:**
- رسالة خطأ: "Error: EADDRINUSE: address already in use"
- التطبيق لا يستجيب

**الحل:**
```bash
# Linux/Mac
lsof -i :3000
kill -9 <PID>

# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F
```

### المشكلة: فشل ارتباط قاعدة البيانات

**الأعراض:**
- "Unable to connect to database"
- رسالة ECONNREFUSED

**الحل:**
1. تأكد من بدء الخادمة:
```bash
# PostgreSQL
sudo systemctl start postgresql

# SQLite - لا يرع بدء
```

2. تإكد مع DATABASE_URL في .env

3. اعادة المحاولة

### المشكلة: بطء في معالجة الملفات

**الأعراض:**
- فشل دائم لرفع الملفات
- رسالة "File too large"

**الحل:**
1. تأكد من الحد الأقصى:
```bash
MAX_UPLOAD_SIZE=52428800  # 50MB
```

2. ترقية صلاحيات الملف

3. تماما تفراغ ملف /tmp

## مشاكل المعالجة

### المشكلة: لا نتائج عند التحويل

**الأعراض:**
- رسالة فارغة
- نظام وقف عند 0%

**الحل:**
1. جرب ملفات ما بعرض مختلفة
2. تأكد من اتصال API
3. فعل وضع DEBUG

### المشكلة: ذاكرة رامة (بطء الدالة)

**الأعراض:**
- رسالة "Out of memory"
- البرنامج يتوقف

**الحل:**
1. زيادة رام النظام
2. قطع المهام الطويلة
3. مراقبة العمليات

## مشاكل N8N

### المشكلة: لا إمكان الاتصال ببانل N8N

**الحل:**
1. تأكد من معرف N8N_HOST
2. تحقق من هوية N8N
3. فعل الوچا في firewall

## مشاكل Authentication

### المشكلة: توكن JWT غير صالح

**الحل:**
```bash
# مرالروج للتوليد المفاتيح

# مشهور JWT_SECRET
npm run generate:secrets

# مزم رآس بقيم جديدة ل .env
```
