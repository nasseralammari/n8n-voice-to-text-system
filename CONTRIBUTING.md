# إرشادات المساهمة

## الترحيب بك!
شكراً لاهتمامك بالمساهمة في مشروع N8N Voice-to-Text System. نحن نرحب بمساهمات الجميع.

## كيفية المساهمة

### 1. Fork المشروع
- انقر على زر Fork في أعلى الصفحة
- اختر حسابك كمقصد

### 2. Clone المشروع المفروع
```bash
git clone https://github.com/YOUR_USERNAME/n8n-voice-to-text-system.git
cd n8n-voice-to-text-system
```

### 3. إنشاء فرع جديد
```bash
git checkout -b feature/your-feature-name
```

### 4. إجراء التغييرات
- قم بإجراء التغييرات المطلوبة
- اتبع معايير الكود الموجودة
- أضف تعليقات واضحة عند الحاجة

### 5. اختبار التغييرات
- تأكد من اختبار جميع التغييرات
- تأكد من عدم وجود أخطاء

### 6. Commit التغييرات
```bash
git add .
git commit -m "وصف واضح للتغييرات"
```

### 7. Push إلى الفرع
```bash
git push origin feature/your-feature-name
```

### 8. إنشاء Pull Request
- اذهب إلى المشروع الأصلي
- انقر على "New Pull Request"
- اختر الفرع الخاص بك
- أضف وصفاً واضحاً للتغييرات
- انقر على "Create Pull Request"

## معايير الكود
- استخدم أسماء متغيرات واضحة ومعبرة
- أضف تعليقات للأجزاء المعقدة
- اتبع معايير PEP 8 للكود Python
- تأكد من أن الكود يعمل قبل الإرسال

## تقارير الأخطاء
إذا وجدت خطأ، يرجى:
1. التحقق من أنه لم يتم الإبلاغ عنه من قبل
2. تقديم تقرير مفصل مع خطوات الاستنساخ
3. تضمين إصدار النظام والمتطلبات

## الأسئلة والنقاش
للأسئلة العامة والنقاشات، يرجى استخدام:
- GitHub Issues للمشاكل التقنية
- المنتدى للأسئلة العامة

شكراً لمساهمتك!
